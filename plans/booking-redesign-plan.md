# خطة تعديل صفحة Booking لتناسب الصفحة الرئيسية

## 📌 النظرة العامة على المشروع
الصفحة الرئيسية [Home.tsx](pages/client/Home.tsx) تمتاز بتصميم حديث وذو صورة احترافية باستخدام خط IBM Plex Sans Arabic واللون الأخضر (#079455) كألوان أساسية، مع تصميمات مرسومة بدقة للعناصر (الرؤوس، الأزرار، الفورمات، الخلفيات). بينما صفحة Booking [Booking.tsx](pages/client/Booking.tsx) تستخدم تصميم أساسي جداً بتنسيق Bootstrap البسيط.

## 🎯 أهداف التعديل
1.統一 الخضوط (Design Language) بين الصفحات
2. تحسين تجربة المستخدم (User Experience)
3. إضافة عناصر تصميم احترافية إلى صفحة Booking
4. الالتزام بأسلوب الصفحة الرئيسية

## 🔍 نقاط التشابه والاختلاف
| العنصر                | الصفحة الرئيسية (Home)                          | صفحة Booking                                     |
|------------------------|-------------------------------------------------|--------------------------------------------------|
| الخط                   | IBM Plex Sans Arabic (wght: 400,500,600,700)     | Bootstrap Default (Arial/Helvetica)              |
| الألوان الأساسية       | الأخضر (#079455), الأسود (#181d25), الأشهر (#667085) | Bootstrap Default (أزرق، أنارنج)                 |
| تصميم الفورمات         | مربع إدخال مع حواف دائرية، ظلال خفيفة           | مربع إدخال بسيط بالحدود الصفراء                  |
| أزرار التفاعل          | أزرار بحجم مناسب، تأثيرات hover                | أزرار بتنسيق Bootstrap بسيط                      |
| الهوامش والتطوير       | تخطيط مرتب بالحجمات الصحيحة                     | تخطيط مع بادنغ ثابتة                             |
| تصميم اللوحة            | غير متوفر في Home                               | عرض لصورة اللوحة البسيطة                         |

## 📊 نماذج الكود المُتوقع للتغيير

### 1. تضمين الخط
```css
@import url('https://fonts.googleapis.com/css2?family=IBM+Plex+Sans+Arabic:wght@400;500;600;700&display=swap');

.booking-page {
  font-family: 'IBM Plex Sans Arabic', Arial, sans-serif !important;
  background: #fff;
  color: #181d25;
  line-height: 1.6;
}
```

### 2. تصميم الفورمات
```css
.booking-form input {
  width: 100%;
  height: 50px;
  border: 1px solid #9DA4AE;
  border-radius: 8px;
  padding: 0 16px;
  font-size: 16px;
  font-family: inherit;
  background: #fff;
  text-align: right;
  color: #000000;
}

.booking-form input:focus {
  outline: none;
  border-color: #079455;
  box-shadow: 0 0 0 3px rgba(7, 148, 85, 0.1);
}
```

### 3. تصميم الأزرار
```css
.booking-btn {
  background: #079455;
  color: #fff;
  padding: 12px 32px;
  border-radius: 8px;
  font-size: 16px;
  font-weight: 600;
  border: none;
  cursor: pointer;
  transition: background 0.3s ease;
  width: 100%;
}

.booking-btn:hover {
  background: #067a47;
}
```
